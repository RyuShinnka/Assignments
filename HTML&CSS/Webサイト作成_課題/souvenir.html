お土産ガイド

白い恋人（北海道）

北海道の定番お土産。ホワイトチョコをラングドシャクッキーで挟んだお菓子で、さくっとした食感と優しい甘さが特徴です。箱入りで配りやすいので、職場や友人へのお土産としても人気です。

公式サイトへ

八ツ橋（京都）

ニッキの香りが豊かな和菓子で、特に生八ツ橋はしっとりとしたもちもちの食感が魅力です。定番のあんこ入りから、抹茶や季節限定フレーバーもあり、京都観光のお土産として親しまれています。

公式サイトへ

もみじ饅頭（広島）

もみじの形をしたカステラ生地で、あんこやクリーム、チョコレートなどを包んだ和菓子。広島の名物で、宮島観光の思い出にぴったりです。現地では揚げもみじ饅頭も人気です。

公式サイトへ

東京ばな奈（東京）

バナナ風味のカスタードクリームをふわふわのスポンジで包んだ洋菓子で、東京土産の代表格です。バリエーションも豊富で、バナナキャラメル味や抹茶味などが販売されています。
